# Video frames extractor
---
Convert your video to a series of images.

Clone it in your local machine
```
git clone _repolink_
```
* Run for program for intructions
```
python3 main.py -h
```
**or**
```
chmod +x main.py
```
```
./main.py -h
```

## Dependencies
---

```
pip3 install requirements.txt
```
